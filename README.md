# r203 tp1 Bulls and Cows
Dans ce projet, nous devloppepons un jeux qui s'appelle Bulls and Cows.

## Description
Ce projet consiste à développer un jeu dans lequel l’utilisateur doit deviner une suite de 4 chiffres, chacun compris entre 0 et 9.
Le programme génère aléatoirement cette combinaison secrète, puis l’utilisateur propose une suite de 4 chiffres pour tenter de la deviner.
Après chaque tentative, le programme indique :

- le nombre de Bulls, correspondant aux chiffres corrects placés au bon endroit ;

- le nombre de Cows, correspondant aux chiffres corrects mais placés au mauvais endroit.

## Fonctionnalités
La fonctionnalité principale de ce jeu est de permettre à l’utilisateur de proposer une suite de 4 chiffres afin de tenter de deviner la combinaison secrète générée aléatoirement par le programme, qui lui indique ensuite le nombre de Bulls (chiffres corrects et bien placés) et de Cows (chiffres corrects mais mal placés).
